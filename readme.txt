== Description ==

Avoid going on Google Maps each time and copying the embed code from there. Just wrap your location within [map] and [/map] and see your map anywhere on Wordpress. Also supports width and height. 

== Installation ==

1. Upload to the "simply-map-me" directory to `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start mapping yourself by wrapping your location with [map][/map] anywhere on your site. 



== Frequently Asked Questions ==

This is super simple plugin so, I don't think any questions will be there. However, you are always welcome to ask/suggest to make things better.

= Maps are not being shown for some of the addresses? =

Please checkout whether those contain some unsupported characters by Google Maps. Example: #, *, etc. You might have written 'Street #' instead of 'Street number' or simply, 'st. '.
